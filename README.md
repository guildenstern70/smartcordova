# Smart Cordova

A demo app with Apache Cordova, Backbone, Zurb and NunJucks

## Frameworks needed (npm install -g)

- Bower 
- Gulp

## JavaScript libraries needed (bower install)

- Zurb foundation
- NunJucks
- Backbone
